---
name: Blog post categories
description: Airtable Blog Posts table has specific category options — only use these when writing blog posts
type: feedback
---

Blog post categories must match Airtable multi-select options exactly:
- Guides
- Tips
- Resources
- News
- Family Caregiving
- Care Types
- Financial Planning

**Why:** Airtable multi-select fields reject values not in the dropdown. Using unlisted categories causes import errors.
**How to apply:** When writing blog posts, only use categories from this list in the frontmatter.
